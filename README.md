## Installation

XXDeviceTool is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby

pod 'XXDeviceTool',:git => 'https://github.com/XXCommonTools/XXDeviceTool.git'

```
